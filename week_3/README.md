Praktikum 1: Menerapkan Control Flows ("if/else")

Langkah 1:
Ketik atau salin kode program berikut ke dalam fungsi main().

```
void main() {
  String test = "test2";
if (test == "test1") {
   print("Test1");
} else if (test == "test2") {
   print("Test2");
} else {
   print("Something else");
}

if (test == "test2") print("Test2 again");
}
```
